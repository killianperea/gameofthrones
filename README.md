# gestion-de-tronos-middleware

Juego de gestión basado en el aclamado libro y posterior serie de televisión Juego de tronos.

# How to run in localhost

## server
in the root folder

    npm install

and then

    node server.js

## client
in 'www' folder

    npm install
    bower install

and then

    gulp

remember to have installed MongoDb and run mongod
